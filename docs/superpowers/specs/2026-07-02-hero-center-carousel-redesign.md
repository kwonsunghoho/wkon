# 히어로 센터형 캐러셀 재설계 — 디자인 스펙

작성일: 2026-07-02
대상: `index.html`의 히어로 섹션(`.hero-scene`) + `Before/After` 섹션(톤 정리)

## 배경 / 문제

기존 히어로는 4개 슬라이드(`hs-slide-0~3`)가 각각 완전히 다른 커스텀 일러스트(마이크+웨이브폼, 뷰파인더, 메트로놈+오선+음표, 바운스 키워드+다이어그램)와 배경 톤을 가지고 있어, 스크롤·전환 시 통일감이 없고 아래 섹션들과도 톤이 끊긴다. 전환은 제자리 크로스페이드(opacity)라 4개를 다 보기 어렵다.

## 목표

4개 슬라이드를 **하나의 공통 뼈대(영상 + 밝은 크림톤 오버레이 + 텍스트)** 로 통일하고, **밝은 웜톤 센터형 캐러셀**로 바꿔 통일감과 탐색성을 높인다.

## 결정 사항

### 톤
- 어두운 계열 사용 안 함. 전체를 밝은 크림/페이지 톤(Warm Sunrise 팔레트)으로.
- 카드 안: mp4 루프 영상 배경 위에 **화면 전체를 덮는 크림톤 반투명 오버레이**(`--bg2` #FFF3E6, 약 72% 불투명도) + 하단으로 갈수록 짙어지는 크림 그라데이션. 영상이 어두워도 전체 톤은 항상 밝게.
- 텍스트: `--text`(#241A12), 강조 단어만 `--action`(#C9471E) 코랄. CTA 버튼은 카드 안.
- 카드 바깥 배경: `--lavender`(#F0D9BC, 살짝 짙은 베이지)로 카드를 은은하게 띄움.

### 레이아웃 — 센터형 캐러셀
- 활성 카드가 화면 중앙, 이전/다음 카드가 좌우에 걸쳐 보임(뷰포트 폭 꽉 채움; 화면 가장자리에서 자연스럽게 잘림).
- 옆 카드: 불투명도 약 0.7, 약간 작게, **각각 바깥으로 ±4도 기울임**(왼쪽 반시계, 오른쪽 시계 방향).
- 카드 비율은 세로형(약 1:1~4:5), 참고 이미지 기준.
- 하단 컨트롤: 기존 점 인디케이터 제거 → **좌우 화살표 버튼 쌍**. 모바일 스와이프도 지원.

### 전환 애니메이션
- 크로스페이드 대신 `transform`(translateX + rotate + scale) + `opacity`를 하나의 easing 곡선(`cubic-bezier`/`ease-out` 계열)으로 묶어 부드럽게 슬라이드.
- 옆 카드가 중앙으로 오면 기울기 0도로 펴지고 불투명도 1·크기 확대. 뚝뚝 끊기지 않게.
- `prefers-reduced-motion`에서는 애니메이션 없이 즉시 전환.

### 영상
- 4개 모두 `<video autoplay muted loop playsinline preload="none">` 무음 루프.
- 실제 촬영본이 아직 없으므로 **스톡 영상/placeholder로 임시 배치**, `src`만 바꾸면 교체되는 구조.
- 영합각의 기존 유튜브 iframe(YT IFrame API 재생추적 포함)도 제거하고 mp4 루프로 통일.
- 활성 슬라이드만 재생, 나머지는 정지(성능).

### 제거 대상 (기존 히어로 마크업/CSS/JS)
- 슬라이드0: 마이크 SVG(`hs-mic-svg`), 펄스링(`hs-rings`/`hsr`), 웨이브폼 32바(`hs-wf`/`hswb`)
- 슬라이드1: 뷰파인더 원/코너/크로스(`hs-vf-*`), REC 인디케이터, 유튜브(`hs-yt-*`, `_ytPlayer`, `_ytPlaying`, YT IFrame API)
- 슬라이드2: 메트로놈(`hs-metro*`), 오선(`hs-staff`/`hs-stline`), 음표(`hs-snotes`/`hs-snote`)
- 슬라이드3: 바운스 키워드(`hs-ans-kw`), 다이어그램 SVG(`hs-ans-*`)
- 공통 CTA바(`hs-cta-bar`) → 카드별 CTA로 대체, 점 인디케이터(`hs-dots`/`hs-dot`) → 화살표

### 유지
- 전/후 토글 플레이어, 피드백 말풍선, 각 챌린지 오디오(보신각/스피닝 전·후 클립).
- `heroSwitchTo(idx)` 진입점 개념은 유지하되 내부를 캐러셀 transform 방식으로 재작성.
- 자동전환(5초) 타이머는 유지하되 오디오/영상 재생 중엔 정지.

### Before/After 섹션 (가벼운 터치)
- 카드 포맷(오디오 플레이어 바 vs 유튜브 썸네일)은 그대로.
- 배경색·카드 테두리·타이포 웨이트만 히어로의 새 크림톤과 자연스럽게 이어지도록 조정. 구조 변경 없음.

## 접근성 / 성능
- `autoplay muted loop playsinline`으로 iOS 자동재생 제약 회피, `poster`로 로딩 전 대체.
- `prefers-reduced-motion`: 영상 정지(첫 프레임/poster), 캐러셀 즉시 전환.
- 활성 카드만 영상 재생.
